Find → Execute → Verify → Memorize → Repeat

A simple 5-step workflow for autonomous AI systems:

1. **Find** — Identify the next task or objective.
2. **Execute** — Perform the task.
3. **Verify** — Independently validate the result. Ideally this is done by a different model, tool, or verification process than the one that executed the task.
4. **Memorize** — Store relevant context, decisions, and outcomes.
5. **Repeat** — Continue the loop until the objective is complete.

```
   FIND
     |
  EXECUTE
     |
   VERIFY
     |
  MEMORIZE
     |
   REPEAT
     |
     └──────→ FIND
```

**Key idea:** Reliable AI agents are built on reliable loops, not better prompts. The verification step should be independent from execution whenever possible, preventing an agent from marking its own work as correct without scrutiny.
